Resolvi com fallback `?? ''` no Translator. O mesmo para `ViolationReport.actionsTaken`. Alternativa seria tornar esses campos nullable nos UI models — a escolha depende de como a UI os consome.

2. **Os `fromJson` factories foram mantidos** nos UI models para backward compatibility com codigo existente que os usa (testes legados, widgets que recebem JSON de APIs).
