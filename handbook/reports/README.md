# Reports — frontend (Conecta Raros)

Registros historicos de sessoes, decisoes e evolucao do projeto.

---

## Formato

Cada report segue o padrao:

```
# Report — <data> — <titulo>

## Contexto
Breve descricao do que motivou a sessao.

## Decisoes Tomadas
Lista de decisoes com justificativa.

## Artefatos Produzidos
Arquivos criados ou modificados.

## Proximos Passos
O que vem a seguir.
```
