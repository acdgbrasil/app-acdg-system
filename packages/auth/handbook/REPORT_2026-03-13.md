# Report — 2026-03-13 — Refatoração do package `auth`

## Contexto

O package `auth` havia sido extraído de `packages/core` com uma estrutura plana (`lib/src/*.dart`) e alguns problemas identificados em 5 code reviews realizados pelo Gabriel. Esta sessão aplicou todos os reviews e reorganizou o package por completo.

### 3. Extração do `OidcClaimsParser`

Lógica de conversão JWT → modelos de domínio extraída de `OidcAuthService` para uma classe com métodos `static` puros:

- `OidcClaimsParser.userFromClaims({uid, claims})` — recebe `Map<String, dynamic>`, retorna `AuthUser`
- `OidcClaimsParser.tokenFromRaw({accessToken, refreshToken, idToken, expiresAt})` — retorna `AuthToken`

Benefício: testável sem `package:oidc`, sem mock, sem inicializar `OidcUserManager`.
